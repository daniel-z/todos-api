Welcome to the ToDos API

  • GET - all ToDos as JSON: /api/todos
  • GET - one ToDo as JSON: /api/todos/:id
  • POST - one ToDo: /api/todos
  • PUT - /api/todos/:id


GET

/api/todos
To GET all tasks
returns: JSON Object 

        
[{
    id: 83515750,
    description: "A new task",
    completed: false
},
{
    id: 1098762,
    description: "other task",
    completed: false
}]

POST

To Post a ToDo task send a json with the next attributes:

  • description: < String >
  • completed: < Boolean >
example: 
        
{
  description: "A new task",
  completed: false
}
returns: JSON Object 
        
{
    id: 83515750,
    description: "A new task",
    completed: false
}

PUT

/api/todos/:id
To Update a ToDo task send a json with the next attributes:

  • description: < String >
  • completed: < Boolean >
example: 
        
{
  description: "A new task",
  completed: false
}
returns: JSON Object 
        
{
    id: 83515750,
    description: "A new task",
    completed: false
}
