Welcome to the ToDos API

  • GET - all ToDos as JSON: /api/todos
  • GET - one ToDo as JSON: /api/todos/:id
  • POST - one ToDo: /api/todos


POST

To Post a ToDo task send a json with the next attributes:

  • description: < String >
  • completed: < Boolean >
example: 
        
{
  description: "A new task",
  completed: false
}
returns: JSON Object 
        
{
    id: 6,
    description: "A new task",
    completed: false
}
